# HOPS Archive

This folder preserves retired HOlomon Photometric Software (HOPS) presentation assets and templates that LEAPS no longer loads or packages.

Archived from the repository root:

- `logo.jpg` — legacy HOlomon Photometric Software logo.

Archived from `hops/files/`:

- `logo.jpg`, `holomon.gif`, and `close.gif` — legacy Tkinter interface artwork.
- `log.yaml`, `log_profile.yaml`, and `log_user.yaml` — legacy HOPS interface configuration templates.
- `reduction_output_description.txt`, `photometry_output_description.txt`, and `fitting_output_description.txt` — legacy output-help text.

The active `hops/` Python package remains at the repository root because LEAPS still imports its star detection, plate solving, alignment, and PyLightcurve fitting routines. The compatibility launcher and upstream license also remain active and are not part of this archive.
